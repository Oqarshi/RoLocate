<h2>🧠 What is RoLocate?</h2>

<p><strong>RoLocate</strong> is a Roblox userscript providing features like server filtering and SmartSearch to find the best servers faster.</p>

<h2>🚀 Features</h2>

<h3>⚙️ General Tab</h3>
<ul>
  <li>🔍 <strong>SmartSearch</strong> – Instantly search for games, users, and groups right from the Roblox search bar.</li>
  <li>🌍 Auto Server Regions – Shows enhanced servers with region and ping info.</li>
  <li>⚡ Fast Server Search – Loads server lists up to <strong>100×</strong> faster by optimizing assets.</li>
  <li>🔄 Invert Player Count – View lowest or highest population servers first.</li>
  <li>🕓 Recent Servers – Quickly rejoin servers you’ve visited in the last 3 days.</li>
</ul>

<h3>🎨 Appearance Tab</h3>
<ul>
  <li>🎬 Disable Trailer Autoplay – Stops trailers from auto-playing.</li>
  <li>📍 Smart Join Popup – Shows server region and details before joining.</li>
  <li>❌ Remove Roblox Ads – Removes most ads across Roblox.</li>
  <li>🔄 Restore Classic Terms – Changes words like “Connections” → “Friends”.</li>
  <li>🧭 Quick Nav – Adds customizable navigation shortcuts.</li>
  <li>🖼️ Custom Backgrounds – Lets you style the Roblox site your way.</li>
</ul>

<h3>🚀 Advanced Tab</h3>
<ul>
  <li>🛠️ Enable Console Logs – See what the script’s doing in your browser console.</li>
  <li>✅ Enable Server Filters – Turn on the advanced filtering UI on game pages.</li>
  <li>🔁 Enable Server Hop Button – Instantly hop to a new game server.</li>
  <li>🔔 Enable Notifications – Get in-browser alerts for updates and actions.</li>
  <li>📌 Set Default Location – Manually choose a fallback region.</li>
</ul>

<h3>✨ Extra Tab</h3>
<ul>
  <li>🧹 Game Quality Filter – Hide low-quality games automatically.</li>
  <li>🧑‍🤝‍🧑 Mutual Friends – View mutual friends on user profiles.</li>
  <li>💬 Disable Chat – Completely remove chat from Roblox.</li>
  <li>🚀 Quick Launch Games – Start your favorite games instantly.</li>
  <li>👋 Show Old Greeting – Bring back the classic Roblox homepage greeting.</li>
  <li>👤 Better Friends – Redesigns the friends list and adds Best Friends.</li>
</ul>
